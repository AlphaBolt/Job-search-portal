# Job Portal
Website to find all job/startup-related job postings across various websites in one place.


## Run Locally

Clone the project

```bash
  git clone https://github.com/AlphaBolt/Job-search-portal.git
```

Go to the project directory

```bash
  cd job-portal
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
